# rin

The template for ruby on rails.  
This application using postgresql.

```sh
$ rails new sample -m https://raw.github.com/everysick/rin/master/app_template.rb -d postgresql
```

## installs

### gems
+ pg  
Application main database.
  
+ redis, redis-rails  
Sub database.
  
+ devise  
Account controller.
  
+ lazy_high_charts  
To edit charts.
